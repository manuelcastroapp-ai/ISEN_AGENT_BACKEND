# MCP Server (ISEN IDE)

Este servidor MCP expone agentes, herramientas y recursos del IDE via stdio.

## Ejecutar

```bash
npm run mcp
```

## Recursos principales

- `ide://workspaces`
- `ide://projects`
- `ide://marketplace`
- `ide://workspaces/{id}/{path}`

## Tools

- `agent.run`
- `tool.run`
- `protocol.run`
- `selfdev.scan`
